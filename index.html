Présentation du projet

IRVE Data Explorer est une application web dédiée à l’étude des infrastructures de recharge pour véhicules électriques. Elle permet de visualiser les stations sur une carte, d’explorer les données issues de la base MySQL et de consulter des statistiques simples pour mieux comprendre la répartition des points de recharge.
