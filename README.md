# IRVE Data Explorer - version HTML classique

Cette version ne contient plus React, Vite, npm, Tailwind ou composants avancés.

## Structure

```text
irve-data-explorer-html/
├── index.html
├── visualisation.html
├── statistiques.html
├── clusters.html
├── predictions.html
├── design-system.html
├── assets/
│   ├── css/
│   │   └── style.css
│   └── js/
│       ├── data.js
│       └── script.js
└── php/
    └── README.md
```

## Lancer le site

Méthode très simple : double-cliquer sur `index.html`.

Méthode propre avec VS Code :
1. Installer l'extension **Live Server**.
2. Ouvrir le dossier `irve-data-explorer-html`.
3. Clic droit sur `index.html`.
4. Cliquer sur **Open with Live Server**.

Méthode Wampserver :
1. Mettre le dossier dans `C:\wamp64\www\`.
2. Ouvrir `http://localhost/irve-data-explorer-html/`.

## Remarque

Le dossier `php/` est gardé pour plus tard, par exemple pour connecter une base de données, lancer un script Python ou appeler un vrai modèle IA.
